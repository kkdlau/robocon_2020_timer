# Robocon 2020 Timer

A cross-platform timer and scoreboard which desgins for robocon 2020.

## Supported platform

* iOS
* Android
* Web

## Debugging

To debug this project, install Flutter plugin in VScode, and follow the [guideline](https://flutter.dev/docs/testing/debugging).
